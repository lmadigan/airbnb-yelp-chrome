Airbnb Yelp Overlay

A Chrome Extension, aimed at
improving booking experience
by Lauren Madigan.

  • Download Extension

This extension is aimed at enhancing the booking
experience of Airbnb users by allowing them
to view a Map with the location of the listing
along side nearby Yelp listings .

A vacation rental is only as good as its nearest coffee shop.


Generates Google Map with Location of Listing.

Shows top 20 Yelp Listings in a 1 mile radius.

Allows for more streamlined, eduated booking process.

Features

Simple, easy to use chrome extension for Airbnb Listing pages.

The extension will pop up a map containging the location of the Airbnb listing
as well as 20 of the top Yelp listings within a 1 mile radius.
